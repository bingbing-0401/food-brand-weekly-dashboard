# 外卖品牌经营数据看板（周报自动化）

## 项目简介
为某外卖餐饮品牌设计的周度经营数据看板，用于追踪门店在各平台（美团/饿了么）的关键经营指标。

## 核心特点
- **字段级自动更新**：修改任意统计字段（如目标值、考核周期、门店范围），报表所有汇总数据、转化率、环比、图表**自动刷新**，无需手动调整
- **参数化设计**：日期范围、平台筛选、目标值等均设计为可修改参数，数据源更新后一键刷新即可同步全表
- **公式驱动**：基于 Excel 数据透视表 + GETPIVOTDATA + 动态命名范围，实现字段间自动联动

## 解决的问题
- 日常经营数据分散，难以快速查看周度表现
- 缺乏统一的指标口径（转化率、到手率等）
- 无法直观对比实际 vs 目标
- **传统报表修改字段后需要重做公式，本看板实现自动适配**

## 看板功能
- **周累计指标**：曝光人数、进店转化率、下单转化率
- **目标追踪**：实际 vs 目标、周环比
- **结果指标**：GMV、商家实收、到手率、客单价
- **过程指标**：曝光→进店→下单漏斗、营销费用占比
- **动态更新**：修改目标值或筛选条件后，所有计算结果自动同步

## 指标说明
| 指标 | 计算公式 | 自动更新方式 |
|------|----------|--------------|
| 进店转化率 | 进店人数 / 曝光人数 | 公式自动计算，随源数据刷新 |
| 下单转化率 | 下单人数 / 进店人数 | 公式自动计算，随源数据刷新 |
| 到手率 | 商家实收 / GMV | 公式自动计算，随源数据刷新 |
| 客单价 | GMV / 有效订单 | 公式自动计算，随源数据刷新 |
| 营销占比 | CPC总费用 / GMV | 公式自动计算，随源数据刷新 |
| 周环比 | (本周值 - 上周值) / 上周值 | 基于动态范围自动计算 |
| 目标进度 | 实际值 / 目标值 | 目标值可修改，进度自动更新 |

## 技术栈
- Excel（数据透视表 + 动态命名范围 + 公式联动）
